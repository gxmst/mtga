# TACZ MTGA (Make Turrets Great Again)

一个基于 [TACZ](https://modrinth.com/mod/tacz) 的我的世界炮台模组。本模组旨在提供能够与 TACZ 枪械完美集成的自动化防御方案。

## 🌟 主要特性

*   **枪械挂载**：支持几乎所有 TACZ 枪械。只需空手右键炮台即可挂载手中的武器。
*   **智能交互**：
    *   **右键**：挂载或取下枪械。
    *   **Shift + 右键**：在手中持有升级插件时进行安装。
    *   **Shift + 左键**：在潜行状态下左键点击炮台即可将其拆卸回物品形式。
*   **红石控制**：炮台支持红石信号。当接收到红石信号时，炮台将进入休眠状态，停止扫描和攻击。
*   **机动部署**：炮台可以被推入矿车或船中，实现移动防御平台（Body 旋转会自动跟随头部视角）。
*   **成长系统**：炮台拥有击杀计数。达到 100 杀和 300 杀（老兵级）时会获得显著的属性提升。

## 🛠️ 升级插件

本模组提供了多种可安装的插件来增强你的炮台：

1.  **火控雷达 (Range Upgrade)**：
    *   大幅提升扫描频率和探测距离。
    *   安装后攻击范围翻倍 (x2)。
2.  **水冷系统 (Speed Upgrade)**：
    *   优化散热，提升连续射击性能。
    *   射速提升 20%。
3.  **弹药制造工厂 (Infinite Ammo)**：
    *   实现真正的弹药自由，不再消耗任何物品。
    *   **平衡性调整**：安装后基础伤害降低至 15%。

## 🤖 实体类型

*   **自动化炮台**：标准型号，拥有现代化的外观。
*   **简易木质炮台**：廉价的早期防御方案，性能与标准版一致，但外观更加古朴。
*   **老兵级自动化炮台**（仅创造模式）：预设了 300 击杀数的最强状态炮台。
